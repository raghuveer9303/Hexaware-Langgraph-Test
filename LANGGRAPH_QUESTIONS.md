# LangGraph Documentation Agent — Sample Questions

Use these questions with `python chatbot.py` to test the agent. Each question is designed to be answered from the [official LangGraph Python documentation](https://docs.langchain.com/oss/python/langgraph/overview).

1. What is LangGraph and what problems is it designed to solve?

2. How do you define state, nodes, and edges using the LangGraph Graph API?

3. What is the difference between the Graph API and the Functional API in LangGraph?

4. How does checkpointing and persistence work in LangGraph?

5. How do you add short-term and long-term memory to a LangGraph application?

6. What streaming modes does LangGraph support, and when would you use each?

7. How do you compose multiple graphs using subgraphs?

8. What are interrupts in LangGraph, and how do they enable human-in-the-loop workflows?

9. How can you replay or fork a past execution using time-travel?

10. What is the recommended way to test a LangGraph agent locally before deployment?
